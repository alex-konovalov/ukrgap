Деятельность Украинской группы пользователей GAP

Украинская группа пользователей системы компьютерной алгебры GAP была создана в августе 2001 г. математиками Киевского , Львовского и Запорожского университетов (Украина) и Университета г.Дебрецен (Венгрия). В 2003 г. к ней присоединились Луганский государственный педагогический университет им. Т. Г. Шевченко и Национальный Университет "Киево-Могилянская Академия", в 2006 - Сумский государственный университет (кафедра электроники и компьютерной техники).

Украинская группа пользователей была создана с целью совместной деятельности в следующих основных направлениях: разработка программного обеспечения для системы GAP, пропаганда и распространение системы, техническая поддержка ее пользователей, и интенсификация обмена научной информацией. Некоторые из проектов, в которых участвуют члены группы, перечислены ниже:

  • Разработка программного обеспечения для системы GAP:
    • Пакет LAGUNA для работы с групповыми кольцами
    • Пакет UnitLib - библиотека нормированных мультипликативных групп модулярных групповых алгебр
    • Пакет Circle - присоединенные группы радикальных алгебр
    • Пакет Wedderga - разложение Веддербарна для групповых алгебр
    • Экспериментальный дистрибутив GAP для Windows
  • Пропаганда и распространение системы GAP:
    • Перевод страницы с дистрибутивом и инструкциями по инсталляции системы GAP 
    • Поддержка зеркала дистрибутива системы на ftp-сервере Запорожского государственного университета в 2001-2005 гг.
    • Учебные материалы по применению системы для преподавания алгебры и теории чисел
  • Техническая поддержка пользователей системы GAP:
    • линия консультаций по адресу konovalov <at> member <dot> ams <dot> org
  • Обмен научной информацией, в т.ч. связанной с системой GAP:
    • публикация новостей и др. материалов на сайте группы
