Что это такое?

В качестве альтернативы ручной распаковке -win.zip архивов, Вам предлагается дистрибутив, обеспечивающий стандартную процедуру инсталляции GAP для Windows. Его разработку и поддержку осуществляет Александр Коновалов. Дистрибутив GAP для Windows создан с помощью NSIS (Nullsoft Scriptable Install System) - свободно распространяемого и открытого программного продукта для создания инсталляторов для Windows.


Загрузка

Для инсталляции системы GAP и пакетов для GAP нужно загрузить следующий файл:
  • gap4r7p5_2014_05_24-20_02.exe (325 MB, md5 checksum 355c9c221dfc4acbae8939a09da14c08): Основные компоненты системы и все текущие версии пакетов для GAP (с исполнимыми файлами для Windows)


Инсталляция

  • запустите загруженный файл
  • ознакомьтесь с лицензионным соглашением
  • выберите компоненты для установки
  • укажите каталог для установки (по умолчанию C:\gap4r7).
  • распакуйте файлы в соответствии с выбранной конфигурацией

Выбирая путь для установки GAP, учтите, что он не должен содержать пробелов. Например, Вы cможете использовать GAP в каталоге вида C:\gap4r7, D:\gap4r7p2 или C:\Math\GAP\gap4r7, но не должны устанавливать систему в такие каталоги как, например, C:\Program files\gap4r7, C:\Users\alex\Мои Документы\gap4r7 и т.п.

Учтите, что некоторые пакеты не будут работать в Windows или же будут работать с ограниченной функциональностью, т.к. они предназначены для работы в среде UNIX/Linux. Подробности см. в документации соответствующих пакетов.


Как это работает

Инсталлятор GAP для Windows:
  • распаковывает архивы в указанный для инсталляции каталог (далее - каталог GAPROOT)
  • корректирует bat-файлы в соответствии с каталогом GAPROOT
  • создает ярлыки в главном меню(опционально)
  • сохраняет информацию в системном реестре Windows (т.о., GAP будет отображаться в списке установленных программ)
  • записывает программу для деинсталляции GAP в каталог GAPROOT
  • создает файл gap.ini в каталоге Windows (для хранения версии GAP, каталогов GAPROOT и GAPROOT/pkg)


Как удалить GAP?

Удаление системы GAP можно выполнить любым из трех способов:
  • Панель управления --> Установка и удаление программ --> Удалить GAP
  • Запустив uninstall.exe в каталоге GAPROOT (gap4r7)
  • Запустив "Uninstall GAP" в каталоге GAP главного меню
Любое из этих действие удалит все каталоги и файлы в каталоге GAPROOT, за исключением созданных Вами в каталоге GAPROOT подкаталогов (если такие существуют). Если нужно, Вы можете впоследствии удалить каталог gap4r7 вручную.


FAQ

Q. Как добавить новые компоненты (в частности, пакеты) к существующей инсталляции GAP?
A. запустите инсталлятор еще раз и повторите выбор компонентов, указав все, которые нужны Вам теперь (все ранее установленные компоненты будут при этом перезаписаны).

Q. Могу ли я иметь несколько инсталляций системы GAP на моем компьютере?
A. Конечно, можете. Учтите, что инсталлятор настроит систему только для последней выполненной инсталляции, а GAP из предыдущих инсталляций возможно понадобится запускать вручную.


Возможные проблемы

Дистрибутив GAP для Windows был успешно протестирован в Windows XP, Windows Vista, Windows 7 и Windows 8. Возможно, он также будет работать и в других версиях Windows.
