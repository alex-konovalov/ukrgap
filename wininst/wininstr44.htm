Что это такое?

В качестве альтернативы ручной распаковке архивов, Вам предлагается экспериментальный дистрибутив для инсталляции системы GAP для Windows. Его разработку и поддержку осуществляет Александр Коновалов. Дистрибутив GAP для Windows создан с помощью NSIS (Nullsoft Scriptable Install System) - свободно распространяемого и открытого программного продукта для создания инсталляторов для Windows.


Загрузка

Для инсталляции системы GAP и пакетов для GAP нужно загрузить следующие файлы:
  • gap4r4p12.exe (49 MB): Основные компоненты системы, а также опциональные компоненты tools, htmie и xtom
  • наиболее свежий архив пакетов для системы GAP packages-<timestamp>.exe (56 MB) из каталога ftp://ftp.gap-system.org/pub/gap/windowsinstaller/
Если ранее Вы уже использовали предлагаемый экспериментальный дистрибутив для инсталляции одной из предыдущих версий системы GAP, начиная с GAP 4.4.6, Вы можете перейти к версии 4.4.12 с помощью следующего обновления (процедура обновления аналогична описанной ниже процедуре инсталляции):
  • fix4r4p12.exe (18 MB)


Инсталляция

1. Сначала нужно инсталлировать систему GAP:
  • запустите gap4r4p12.exe
  • ознакомьтесь с лицензионным соглашением
  • выберите компоненты для установки
  • укажите каталог для установки (по умолчанию C:\gap4r4)
  • распакуйте файлы в соответствии с выбранной конфигурацией

Как это работает

Инсталлятор GAP для Windows:
  • распаковывает архивы в указанный для инсталляции каталог (далее - каталог GAPROOT)
  • корректирует bat-файлы в соответствии с каталогом GAPROOT
  • создает ярлыки на рабочем столе, в главном меню и панели быстрого запуска (опционально)
  • сохраняет информацию в системном реестре Windows (т.о., GAP будет отображаться в списке установленных программ)
  • записывает программу для деинсталляции GAP в каталог GAPROOT
  • создает файл gap.ini в каталоге Windows (для хранения версии GAP, каталогов GAPROOT и GAPROOT/pkg)
Инсталлятор пакетов для GAP не выполняет никаких дополнительных действий - он просто распаковывает выбранные пакеты в каталог для пакетов GAP.


Как удалить GAP?

Удаление системы GAP можно выполнить любым из трех способов:
  • Панель управления --> Установка и удаление программ --> Удалить GAP 4.4.12
  • Запустив uninstall.exe в каталоге GAPROOT (gap4r4)
  • Запустив ярлык "Uninstall GAP 4.4.10" в каталоге GAP 4.4.12 главного меню
Любое из этих действие удалит все каталоги и файлы в катлоге GAPROOT, за исключением созданных Вами в каталоге GAPROOT подкаталогов (если такие существуют). Если нужно, Вы можете впоследствии удалить каталог gap4r4 вручную.


FAQ

Q. Как добавить новые компоненты системы GAP?
A. запустите снова gap4r4p12.exe и опять выберите компоненты, указав все, которые нужны Вам теперь (все ранее установленные компоненты будут при этом перезаписаны).

Q. Как добавить новые пакеты в систему?
A. Запустите packages-....exe и укажите те пакеты, которые необходимо добавить. Остальные пакеты будут проигнорированы независимо от того, установлены ли они уже или нет.

Q. Как я могу обновить уже установленные пакеты, заменив их новыми версиями?
A. Конечно же, Вы можете загрузить новую версию объединенного дистрибутива всех пакетов packages-....exe, удалить все подкаталоги (кроме tomlib !!!) из каталога GAPROOT\pkg и установить все пакеты занова. В качестве альтернативы полной переинсталляции всех пакетов, Вы можете загрузить архив нужного Вам пакета с сайта GAP или страницы соответствующего пакета, распаковать его и вручную заменить подкаталог данного пакета в каталоге GAPROOT\pkg его новой версией.

Q. Могу ли я иметь несколько инсталляций системы GAP на моем компьютере?
A. Конечно, можете. Инсталлятор GAP для Windows позволяет установить только одну инсталляцию, а остальные копии GAP могут быть добавлены стандартным образом в соответствии с находящимися здесь инструкциями. Вам нужно будет вручную отредактировать соответствующие bat-файлы и создать ярлыки.


Возможные проблемы

Дистрибутив GAP для Windows был успешно протестирован в Windows 98, Windows 2000 и Windows XP. Выяснено, что в Windows 98 при удалении системы не удаляются автоматически ярлыки на рабочем столе и в панели быстрого запуска, поэтому их нужно затем удалять вручную.

В Windows 2000 и старше Вы должны иметь соответствующие права на выполнение некоторых операций при инсталляции основных компонент системы GAP. Если у Вас нет прав администратора, то Вы можете выполнить инсталляцию GAP стандартным способом в соответствии с находящимися здесь инструкциями.
В Windows Vista инсталляция в каталоге, отличном от принятого по умолчанию C:\GAP4R4 может не работать. Также, для запуска GAP working в окне RXVT, укажите режим совместимости с Windows XP в свойствах ярлыка.
