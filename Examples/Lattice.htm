Построение решетки подгрупп с помощью графического интерфейса ХGAP

Система компьютерной алгебры GAP работает под управлением операционных систем Windows, MacOS, но максимальная эффективность достигается при ее применении в операционной системе Linux. В этом случае можно также установить share package (расширение системы) XGAP и использовать графический пользовательский интерфейс. Ниже помещены скриншоты нескольких экранов XGAP. Для загрузки скриншота в формате PNG с разрешением 800х600 выберите нужный скриншот левой кнопкой мыши.
 
Здесь изображено начало работы с XGAP. XGAP имеет удобную панель меню, позволяет работать в многоконном режиме, когда в других окнах могут быть одновременно запущены текстовый редактор для редактирования текста программы, и руководство по системе GAP Функция GraphicSubgroupLattice(G) строит решетку подгрупп группы G. В данном примере изображена решетка подгрупп группы диэдра порядка 16, заданная командой  G:=DihedralGroup(16). Выбор подгруппы  правой кнопкой мыши открывает окно с ее свойствами 
Вы можете производить различные действия над подгруппами (в зависимости от того, одна или несколько подгрупп выбраны) Другой раздел меню позволяет задавать различные параметры отображения решетки подгрупп на экране

При этом нормальные подгруппы изображаются с помощью ромбов, а те, которые не являются нормальными - с помощью кругов. С помощью мыши взаимное расположение подгрупп может быть изменено, однако классы сопряженных подгрупп всегда изображаются наборами соприкасающихся окружностей.

Изображение решетки подгрупп может быть сохранено в файл и затем распечатано. Вы можете посмотреть полученное таким образом изображение решетки подгрупп симметрической группы S4 в форматах pdf и ps.
